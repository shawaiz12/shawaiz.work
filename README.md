### 🗺️ Procedural Cave Generator
**Unity 6 · C# · Custom Editor Tool · Procedural Generation**

A custom in-editor Unity tool that generates optimised, designer-controlled 3D low-poly cave layouts procedurally. Built to explore data-driven environment generation and editor tooling.

**Technical highlights:**
- Cellular automata simulation engine using the 4-5 rule across multi-pass loops
- Model-View separation — layout computed in memory as a 2D integer array before any rendering occurs
- Draw call optimisation by skipping fully enclosed wall tiles (8-neighbour check)
- Deterministic seed-based generation for reproducible layouts
- Custom Inspector GUI via `OnInspectorGUI` override — no Play mode required for iteration
